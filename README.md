# Doggoscale!

Einfache Waage für Hunde.

# Material
+ 1x Siebträgerplatte
+ 4x Wägezellen
+ 1x Wägezellencontroller HX711
+ 1x Microcontroller ESP8266 ESP-12F
+ 1x Display 2,8 Zoll LCD TFT Touch Display, 320x240 Auflösung, ILI9341 Treiber, SPI
+ 1x Micro-USB Montageanschluss
+ 1x RJ45 Breakout Board
+ Lochrasterplatinen
+ Steckleisten
+ Lötkabel
+ Heißkleber
+ Für Gehäuse: 3D-Drucker mit PLA-Filament
